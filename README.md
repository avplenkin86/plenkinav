# Portfolio - Алексей Плёнкин

Простая статическая веб-страница резюме технического руководителя и ИТ-архитектора.

## Структура файлов

- `index.html` - главная страница
- `style.css` - стили на основе Material Design 3
- `DESIGN.md` - спецификация дизайна (цвета, типографика)
- `potfolio.md` - исходный контент в формате Markdown
- `screen.png` - скриншот дизайна

## Просмотр

Просто откройте `index.html` в любом современном браузере.

## Особенности

- Адаптивный дизайн (работает на мобильных устройствах)
- Темная тема на основе Material Design 3
- Чистый HTML + CSS без зависимостей
- Быстрая загрузка

## Цветовая палитра

- Основной фон: #0b141c
- Текст: #dae4ef
- Акцент: #b6c4ff
- Вторичный текст: #c3c5d9

## Типографика

- Заголовки: Space Grotesk
- Основной текст: Inter
- Моноширинный: системный monospace
